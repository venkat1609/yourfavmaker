# Ethereal Commerce

This repository hosts a premium marketplace experience built with Vite, React, and tailored seller/admin tooling. It powers the YourFavMaker storefront with rich dashboards, Supabase-backed data, and a modern design system.

## Getting Started

1. **Install dependencies**
   ```bash
   npm install
   ```
2. **Run the development server**
   ```bash
   npm run dev
   ```
3. **Build for production**
   ```bash
   npm run build
   ```

## Project Highlights

- **Stack**: Vite + TypeScript + React + shadcn-ui + Supabase
- **Dashboards**: Separate seller, admin, and storefront views with data-driven widgets and forms
- **Custom tooling**: Notification center, product/collection managers, and shared UI primitives under `src/components`

## Contributing

1. Create a feature branch from `main`.
2. Add or update tests as needed (see `vitest.config.ts`).
3. Run `npm run lint` and `npm run test` before opening a PR.
